1.什么是管道符

管道符“|”
命令格式：命令A|命令B，即命令1的正确输出作为命令B的操作对象

2.老板让我打开A服务器的 只知道名称好像含有xxx服务的web界面，请问怎么做？

  1. 得到A服务器的ip
  2. ps -ef|grep xxx --》pid
  3. netstat -nlp|grep pid --》port
  4. 打开浏览器: http://ip:port

3.在杀死进程时，我们应该是否要确认 这个进程就是我们可以杀的？那么全局杀死所有 名称xxx的 进程，命令是什么？

  • 杀死进程时需要确认这个进程时我们可以杀死的
  • 杀死所以名称是xxx的进程：kill -9 $(pgrep -f xxx)

4.netstat -nlp | grep pid 显示： 127.0.0.1:880；请问外部可以访问吗？

外部访问不了，只能本地访问

5.netstat -nlp | grep pid 显示： ip:880；外部无法访问，请问应该要注意什么？

  1. ping ip
  2. telnet ip port
  3. 如果以上都不行网络不通有可能你的服务器 防火墙 开启，云主机 需要开启 安全组策略
    直接找Linux运维 网络工程师 加防火墙(硬件)策略

6.全局模糊搜索名称含有xxx 的文件、文件夹

find / ‘xxx’

7.搜索 安装 卸载 rpm

  • 搜索 rpm -qa|grep xxx
  • 安装 rpm -i xxx
  • 卸载 rpm -e xxx --nodeps

8.谈谈你们对命令帮助 ，应该怎么看呢？会看吗？

  • xxx --help
  • man xxx

9.生效完环境变量文件后，要习惯做一件事，是什么呢？

用which命令确认配置是否成功

10.which的执行结果输出，是来自哪个系统环境变量呢？

来自于$PATH环境变量

11.vi的三种模式

  1. 命令模式
  2. 命令模式下按 i 进入编辑模式
  3. 命令模式下按：进入尾行模式

12.简述vi编辑文件正常流程

  1. vi 打开一个文件进入命令模式
  2. 按 i 进入编辑模式，编写文本
  3. 按ESC退出编辑进入命令模式
  4. 按 :wq保存文本退出或按 :q退出不保存

13.简述vi的如何清空文件内容和粘贴复制内容

  1. gg 跳转到第一行的第一个字母
  2. dG 删除当前及以下所有行
  3. 按 i 键进入编辑模式
  4. 粘贴复制的内容

14.rwxr-x-w- 数字多少

  • rwx: 7
  • r-x: 5
  • -w-:2
  • =752

15.window Linux 互相传输文件，需要安装哪个rpm？

安装：yum install -y lrzsz
sz xxx.log : 从linux下载到windows
rz : 从windows上传到linux
