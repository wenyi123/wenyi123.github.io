1.查看当前目录

pwd

2.隐藏文件、文件夹以什么开始，怎样查看

隐藏文件夹 文件是以.开头
通过ls -l -a命令查看

-l 等价于什么？

等价于ll

4.级联创建文件夹

mkdir -p f1/f2/f3

5.创建文件哪几种方式

1.touch 文件名
2.vi 文件名
3.echo "” > 文件名

6.cp和mv谁快？ 能不能在cp 、mv过程，顺便修改名称呢？

mv是移动，始终一份，快
cp 是复制，两份，慢
可以顺便修改名称

7.查看文件的大小哪两种命令？

ls -ll 以字节为单位显示
ls -lh 以KB、MB等为单位显示

8.查看文件夹的大小的命令？

du -sh

9.ll 命令执行完成后，文件、文件夹展示一长串信息，有哪些？

权限 文件数 用户名 用户组 大小 日期时间 文件名
dr-xr-x—. 6 root root 4096 Apr 15 21:51 file

10.绝对路径和相对路径，区别是什么？

绝对路径：路径的写法一定是由根目录 / 写起的
相对路径：路径的写法不是由根目录 / 写起的，从当前目录写起

11. root的home目录在哪？普通用户的默认家目录在哪？

root的home目录: /root
普通用户的home目录: /home/用户名

12.切换到用户的家目录，哪三种方式？

  1. cd ~
  2. cd /home/用户名
  3. cd 回车

13.切换到 上一层目录 与 上一次目录，命令分别是什么？

切换上一层: cd …/
切换上一次: cd -

14.清除屏幕信息

clear

15.查看文件内容 哪几个命令？

  1. cat 查看全部
  2. more 一页页翻
  3. less 上下键翻
  4. tail 实时监控

16.实时查看文件内容，-f -F区别是什么？

tail
-f 当文件删除或重命名时停止更新
-F 当文件删除或重命名时会尝试重试读取文件

17.谈谈你对如何定位ERROR的理解？

  1. cat xxx.log | grep ERROR 当前行
  2. cat xxx.log | grep -A 5 ERROR 后5行
  3. cat xxx.log | grep -B 5 ERROR 前5行
  4. cat xxx.log | grep -C 5 ERROR 前后各5行

18. > >>区别是什么？

> 创建或覆盖文件
>> 追加内容到文件

19.设置变量 key= value ,这样写对吗？如何引用呢？

设置变量 export key=value
查看变量 echo $key

20. 全局环境变量文件在哪？ 个人环境变量文件(推荐哪个)在哪？如何生效？

全局 : /etc/profile

个人: ~/.bash_profile
~/.bashrc 推荐

生效: source ~/.bashrc
