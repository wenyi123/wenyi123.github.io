查看历史命令：history

用户组信息 ：/etc/group

设置主组

- ruoze,这个命令执行完成后，做哪些事？

切换用户到ruoze，cd到home目录下执行环境变量文件

10.一个普通用户，想要瞬间临时获取root的最大权限，请问在哪个配置文件修改什么内容？ 在执行命令时，sudo命令加在前面？

修改/etc/sudoers文件
用户名 ALL=(root) NOPASSWD:ALL

sudo 写在命令前面

11.> >>区别是什么

> 创建或覆盖文件
>> 追加内容到文件

12.rwx分别代表数字多少？ rwx- - x - w - ,请问权限数字多少？

r:4；w:2；x:1

rwx- - x - w -: 712

13.上题中，三组分别代表哪个方面的权限呢？

r: 可读；w: 可写；x: 可执行

14.修改权限 和 所属用户用户组的命令，分别是什么？ 假如修改文件夹，加什么参数

修改权限: chmod 文件
修改所属用户用户组: chown 用户:用户组 文件

修改文件夹权限加-R参数

15.简述vi编辑文件的流程

  1. vi 文件名
  2. 按i键进入编辑模式
  3. 按Esc进入命令模式
  4. :wq保存编辑退出
  5. :q! 不保存强行退出

16.简述vi清空文件的命令

  1. 在命令模式按gg–》dG
  2. cat /dev/null > 文件名
  3. true > 文件名
  4. echo “” > 文件名 会有一个byte的残留

17.vi编辑文件，想要从外部文件内容粘贴复制到这个文件，必须提前做一件什么事？否则数据丢失

必须进入编辑模式

18.查看进程、端口号命令？

查看进程: ps -ef|grep 应用名
查看端口号: netstat -nlp|grep pid

19.window 和 Linux 去测试端口号的命令，会部署吗？会使用吗？

安装测试网络工具: yum install -y net-tools.x86_64

测试ip: ping ip
测试端口号: telnet ip port

20.netstat 一个服务时，显示 127.0.0.1:8899 ，外部window 或者其他服务器进行 访问，能通吗？

外网不能访问，只能本地访问

21.netstat 一个服务时，显示 hostname:8899 ，外部window 或者其他服务器进行 访问，假如不通，我们应该要调整什么？

有可能你的服务器 防火墙 开启，云主机 需要开启 安全组策略
或者直接找Linux运维 网络工程师 加防火墙(硬件)策略
